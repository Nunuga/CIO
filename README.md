# Hajime Biography Demo

Минимальный Next.js + TypeScript + Tailwind + three.js проект,
повторяющий идею страницы `/biography` с сайта hajimewatanabe.jp:

- полноэкранный шейдерный фон на three.js;
- горизонтальный скролл биографии;
- индикатор прогресса внизу.

## Запуск

```bash
npm install
npm run dev
```

Открой `http://localhost:3000/biography`.
